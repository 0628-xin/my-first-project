# 百度竞价落地页 URL 匹配工具

把百度竞价账户里的关键词批量匹配到对应的落地页 URL。本地离线运行，双击即用。

## 快速开始

1. 用浏览器打开 `tool.html`
2. 「域名管理」Tab 添加落地页域名
3. 「规则配置」Tab 添加匹配规则
4. 「匹配执行」Tab 粘贴关键词，执行匹配，复制结果

## 功能

- 10 种匹配类型：关键词精确/包含/正则、单元精确/包含、计划精确/包含、计划+单元组合、类别精确/包含
- 多级优先匹配：规则按顺序从上到下，命中即停
- 多域名管理
- 变量替换：`{{kw}}` `{{plan}}` `{{unit}}` `{{category}}`
- 批量输入：支持 Excel 直接粘贴（计划+单元+关键词 三列）
- 结果导出：复制回 Excel 或导出 CSV
- 规则导入导出：JSON 格式备份

## 文件

- `SKILL.md` — Skill 定义文件，AI 助手读取后按流程引导用户
- `tool.html` — 实际工具，浏览器打开即用

## 从真实账户数据构建规则

当有百度竞价导出的关键词 CSV 时，可以让 AI 助手分析账户结构并自动生成规则 JSON：

1. 提供关键词 CSV 文件
2. AI 分析域名/计划/单元/URL 规律
3. 生成 `<账户名>_匹配规则配置.json`
4. 在工具中「导入规则」加载

详见 SKILL.md 中的「路径 B」流程。
