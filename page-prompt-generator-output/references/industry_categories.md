# 4级行业分类体系

适用于政策/招商页面的行业分类数据。3-4级可多选。

## 一级：行业大类（6个）

1. 制造业
2. 信息传输、软件和信息技术服务业
3. 科学研究和技术服务业
4. 金融业
5. 商务服务业
6. 现代物流业

## 二级：行业中类

### 制造业下
- 通用设备制造业
- 专用设备制造业
- 汽车制造业
- 电子信息制造业
- 新材料制造业
- 食品制造业
- 纺织服装制造业
- 化工制造业
- 钢铁制造业
- 有色金属制造业

### 信息传输、软件和信息技术服务业下
- 软件开发
- 信息技术服务
- 互联网服务
- 数据处理
- 电信服务

### 科学研究和技术服务业下
- 研发与试验发展
- 专业技术服务
- 科技推广服务
- 工程技术服务

### 金融业下
- 银行业
- 保险业
- 证券业
- 基金业
- 金融科技

### 商务服务业下
- 法律服务
- 会计审计
- 咨询服务
- 广告服务
- 会议展览

### 现代物流业下
- 仓储服务
- 配送服务
- 冷链物流
- 跨境物流
- 供应链管理

## 三级：行业小类（示例）

### 汽车制造业下
- 汽车整车制造
- 汽车零部件制造
- 新能源汽车制造

### 电子信息制造业下
- 通信设备制造
- 计算机制造
- 半导体集成电路制造
- 电子元器件制造
