# 页面风格配置

6种专业页面风格，每种包含风格特征、适用场景和推荐搭配。

## 1. 专业政务

- **风格特征**：沉稳大气、层次分明、信息密集但有序、权威可信
- **设计要素**：深色导航、大量内容区块、数据图表、公文式排版
- **适用场景**：政策解读、税务咨询、政务公开、行业报告
- **推荐搭配色系**：深蓝专业、现代简约
- **推荐字体**：思源黑体 / Noto Sans SC（正文），思源宋体 / Noto Serif SC（标题）
- **排版要点**：标题层级清晰（h1-h4），正文14-16px，数据表格规范对齐

## 2. 现代简约

- **风格特征**：简洁克制、留白充分、视觉节奏轻快、功能导向
- **设计要素**：大面积留白、细线条边框、简洁图标、扁平化按钮
- **适用场景**：通用展示、商务服务、信息门户、产品介绍
- **推荐搭配色系**：现代简约、科技蓝调
- **推荐字体**：Inter / 思源黑体（正文），14-16px，行距1.6
- **排版要点**：卡片式布局、图标+文字组合、信息分块清晰

## 3. 科技感

- **风格特征**：前沿尖端、数据可视化、动态交互、沉浸体验
- **设计要素**：渐变背景、粒子效果、数据仪表盘、3D元素、光效
- **适用场景**：科技园区、创新平台、智能制造、数字化展示
- **推荐搭配色系**：科技蓝调、深蓝专业
- **推荐字体**：思源黑体 + 英文混合，标题加粗18-24px
- **排版要点**：数据可视化为主、动画过渡流畅、信息密度适中

## 4. 高端商务

- **风格特征**：奢华精致、细节考究、品牌气质突出、尊贵体验
- **设计要素**：大图背景、精致排版、金色点缀、高端摄影素材
- **适用场景**：总部经济、投资推广、VIP服务、高端招商
- **推荐搭配色系**：高端紫调、深蓝专业
- **推荐字体**：思源宋体（标题）+ 思源黑体（正文），字号偏大
- **排版要点**：标题醒目、间距宽松、细节装饰考究

## 5. 绿色生态

- **风格特征**：自然有机、清新舒适、生态友好、和谐共生
- **设计要素**：自然色彩、有机曲线、植物元素、环保图标
- **适用场景**：生态园区、环保产业、绿色建筑、农业科技
- **推荐搭配色系**：绿色生态、活力橙调
- **推荐字体**：思源黑体（正文），圆角图标，柔和线条
- **排版要点**：圆润边角、柔和阴影、自然配色、留白较多

## 6. 工业风格

- **风格特征**：硬朗有力、功能明确、结构清晰、效率导向
- **设计要素**：粗线条、几何网格、工业图标、数据面板
- **适用场景**：制造业园区、厂房展示、工业地产、基础设施
- **推荐搭配色系**：现代简约、深蓝专业
- **推荐字体**：思源黑体（粗体标题），14-16px正文
- **排版要点**：网格布局、功能分区明确、信息密度高、硬朗边角
