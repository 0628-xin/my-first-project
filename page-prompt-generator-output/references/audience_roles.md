# 目标受众角色

## 政策/招商页面受众（40个角色）

### 高管层
| 角色 | 核心关注点 | 内容建议 | 专业表达方式 |
|------|-----------|----------|-------------|
| 董事长 | 企业战略方向、产业园区选择、长期发展价值 | 展示战略级产业布局和长远发展规划 | "战略布局"、"产业矩阵"、"长期价值" |
| 董事 | 产业发展趋势、投资回报、合规风险 | 提供行业趋势数据和合规保障说明 | "产业洞察"、"回报预期"、"合规保障" |
| 监事 | 经营合规性、财务透明度、风险防控 | 提供合规报告和风控体系展示 | "合规运营"、"风险管控"、"治理规范" |
| 总经理/CEO | 整体运营效率、资源配置、增长策略 | 展示资源整合能力和运营效率数据 | "资源整合"、"运营效能"、"增长引擎" |
| 合伙人 | 合作模式、利益分配、长期共赢 | 展示合作框架和共赢机制 | "合作共赢"、"利益共享"、"长期伙伴" |

| 财务总监(CFO) | 资金安全、成本控制、投资回报 | 提供财务保障方案和ROI数据 | "资金安全"、"成本优化"、"投资回报率" |
| 财务经理 | 税务筹划、申报流程、财务合规 | 提供税务优化方案和合规指引 | "税务优化"、"合规申报"、"财务规范" |
| 财务专员 | 具体操作流程、表格填写、日常申报 | 提供操作指南和模板下载 | "操作指引"、"模板下载"、"流程规范" |

| 审计专员 | 合规审计、风险排查、整改跟踪 | 提供审计方案和整改流程 | "合规审计"、"风险排查"、"整改闭环" |

| 企业家 | 产业生态位、战略协同、合作模式 | 展示产业生态和合作机会 | "产业生态"、"战略协同"、"合作共赢" |

| 企业董事 | 投资回报、合规风险、战略布局 | 揕供投资分析和合规保障 | "投资回报"、"合规保障"、"战略布局" |
| 股东 | 收益预期、资产安全、增值空间 | 揕供收益预测和资产安全说明 | "收益预期"、"资产安全"、"增值空间" |
| 监事 | 经营合规、财务透明、风险防控 | 展示合规运营和风控体系 | "合规运营"、"财务透明"、"风险防控" |
