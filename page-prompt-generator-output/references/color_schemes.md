# 品牌色系配置

6种专业品牌色系，每种包含色彩哲学、主色调、中性色完整配置和使用场景。

## 1. 深蓝专业

- **定位**：稳重、信任、权威
- **色彩哲学**：深蓝色象征专业可靠、值得信赖，传达稳健经营和长远发展的品牌形象。适合政策推广和高端商务场景。
- **主色调**：
  - Primary: `#1E3A8A` (深蓝) — 主要标题、关键按钮、重要信息区
  - Primary Light: `#3B82F6` (亮蓝) — 辅助高亮、链接、次要按钮
  - Primary Dark: `#1E40AF` (中深蓝) — 导航栏、侧边栏背景
- **中性色配置**：
  - Background: `#F8FAFC` (极淡灰) — 页面背景
  - Surface: `#F1F5F9` (淡灰蓝) — 卡片背景、内容区块
  - Border: `#E2E8F0` (淡灰) — 边框、分割线
  - Text Primary: `#0F172A` (深黑蓝) — 正文文字
  - Text Secondary: `#64748B` (中灰) — 辅助说明文字
  - Text Muted: `#94A3B8` (淡灰蓝) — 时间、标签等
- **适用场景**：政策页面、高端招商、金融财税、政府门户

## 2. 科技蓝调

- **定位**：创新、未来、前沿
- **色彩哲学**：科技蓝传达创新引领、面向未来的品牌气质，适合科技园区和创新产业推广。
- **主色调**：
  - Primary: `#0EA5E9` (科技蓝) — 主标题、数据图表、核心按钮
  - Primary Light: `#38BDF8` (亮天蓝) — hover状态、进度条
  - Primary Dark: `#0369A1` (深蓝) — 导航、重要标签
- **中性色配置**：
  - Background: `#F0F9FF` (极淡蓝白) — 页面背景
  - Surface: `#E0F2FE` (淡蓝) — 卡片背景
  - Border: `#BAE6FD` (中淡蓝) — 边框
  - Text Primary: `#0C4A6E` (深蓝黑) — 正文
  - Text Secondary: `#075985` (中深蓝) — 辅助文字
  - Text Muted: `#0284C7` (蓝) — 标签
- **适用场景**：科技园区、创新孵化、数字平台、智能制造

## 4. 现代简约

- **定位**：简洁、高效、专业
- **色彩哲学**：现代简约以黑白灰为主基调，辅以少量高亮色点缀，传达简洁高效、注重实质的品牌气质。
- **主色调**：
  - Primary: `#18181B` (近黑) — 主标题、核心按钮
  - Primary Light: `#27272A` (深灰) — 导航栏
  - Accent: `#2563EB` (蓝) — 少量点缀，链接、CTA按钮
- **中性色配置**：
  - Background: `#FAFAFA` (近白) — 页面背景
  - Surface: `#F4F4F5` (极淡灰) — 卡片背景
  - Border: `#E4E4E7` (淡灰) — 边框
  - Text Primary: `#18181B` (近黑) — 正文
  - Text Secondary: `#71717A` (中灰) — 辅助
  - Text Muted: `#A1A1AA` (淡灰) — 标签
- **适用场景**：通用页面、商务展示、信息门户、政策平台
